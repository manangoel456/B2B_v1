HOW TO ADD DOCUMENTS
====================

Drop your PDF certificate files into this folder (public/documents/).

File naming convention:
  who-gmp-certificate.pdf          → WHO-GMP Certificate
  cdsco-approval.pdf                → CDSCO Approval Letter
  iso-9001-certificate.pdf          → ISO 9001:2015 Certificate
  drug-license.pdf                  → Drug Manufacturing License
  gst-certificate.pdf               → GST Registration Certificate
  msme-certificate.pdf              → MSME/Udyam Registration

Once you place a file here, it is automatically available at:
  https://barekyne.in/documents/who-gmp-certificate.pdf

The Certifications page at /certifications will show a 
"Download Certificate" button as soon as the file exists.

To mark a document as "Available" on the website:
  - Set file: "who-gmp-certificate.pdf" in the certifications page config
  - The button changes from "Request via WhatsApp" to "Download PDF"
